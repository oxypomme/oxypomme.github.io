Simple MHW Mod Manager

SMHWMM is a simple mod manager for Monster Hunter World

Latest stable version :

Why another mod manager ?

Why download this if a plenty (2 lol) of mod manager already exist ?

Because it's light (less than 2MB to install), it's powerful, fast and simple : You have just to add a mod in the list, activate it and here you go.

And what else ?

You can change rename the mod or it's order in the list, run the game and other vital functions (like a dark mode... not for now but later...)

Installation

Download the latest version, execute it. You can now install mods through SMHWMM !
