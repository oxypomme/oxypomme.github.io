Nim-Mk2

Jeu de Nim en console, avec couleurs et interface utilisateur.

Dernière version stable :

Installation

Télécharger la dernière version
OU
Copiez / téléchargez le repo (cette méthode nécessite de rebuild le projet, les commandes à faire sont ci-dessous).

Linux

                        
$ make

Windows

                        
> mingw32-make
Nécessite d'avoir installé MinGW.

Utilisation

Linux

                    
$ ./bin/nim

Windows

                    
> ./bin/nim.exe
Double cliquer sur executable marche aussi.
